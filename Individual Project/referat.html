CSS: стилове, класове, селектори

Какво е CSS

CSS (Cascading Style Sheets) е език за програмиране, който се използва за описване на представянето на уеб страници. Прилага се, както върху HTML документи, така и върху XML уебстраници и документи. HTML се използва за цялостното структуриране на един уеб документ, докато CSS отговаря за стилизирането на този HTML. Най-често използвани са: промяна на шрифт, цвят и размер на текста, фон на страницата, форматиране на елементите на HTML документа и др.

Употреба и стилове

Както при HTML, CSS се пише в текстов редактор. Съществуват три основни начина, чрез които се задава стил на HTML докуемнта. Това са Inline CSS, Internal CSS и External CSS. Най-главната разлика между Inline CSS и External CSS е това, че Inline стилът се изпълнява по-бързо, защото браузърът тегли само един файл, докато при External стила, браузърът е длъжен да изтегли HTML и CSS файловете отделно. Сега ще разгледаме всеки от трите стила [1]:

Inline CSS:

Inline CSS се използва, за да стилизира специфичен HTML елемент. Най-често се използва с тестова цел. Не се препоръчва използването на този метод. За този стил се използва style атрибут, който се добавя към съответния HTML таг.

                            
                                <!DOCTYPE html>
                                <html>
                                <body style="background-color:black;">

                                <h1 style="color:white;padding:30px;>"Hostinger Tutorials</h1>
                                <p style="color:white;"> Something usefull here. </p>

                                </body>
                                </html>

Предимства на Inline CSS:

  • Може лесно и бързо да се вмъква стилизация към някоя HTML страница. Затова този метод е удобен за тестване и визуализиране на направените промени.
  • Не се изисква създаването на отделен документ, който да съдържа стилизацията.

Недостатъци на Inline CSS:

  • Добавянето на стил поотделно към всеки един от елементите на HTML файла е загуба на време и прави структурата на документа нечетима.
  • Стилизирането на множество елементи по този начин може да повлияе на размера на страницата и времето за изтегляне.

Internal CSS:

Internal CSS изисква да бъде добавен <style> таг в <head> секцията на HTML документа. Този стил е ефективен, когато се ползва за документи от една страница. За стилизиране на документи с множество страници, използването на този стил не е ефективно, защото трябва да се повтаря стилизацията на всяка страница.

Пример за използването на Internal CSS:

  1. Отворете вашата HTML страница и намерете <head> тага.
  2. Поставете следния код след <head> тага:

    3. <style type="text/css">

  3. Добавете CSS правила на нов ред. Пример:
    4. 
                            body {
                                background-color: blue;
                            }

                            h1 {
                                color: red;
                                padding: 60px;
                            }
  4. Напишете затварящ таг:
    5. </style>

Накрая HTML документът ще изглежда по този начин:


                            <!DOCTYPE html>
                            <html>
                                <head>
                                    <style>
                                        body {
                                            background-color: blue;
                                        }
                                        h1 {
                                            color: red;
                                            padding: 60px;
                                        } 
                                    </style>
                                </head>
                                <body>
                                    <h1>Tutorial</h1>
                                    <p>This is our paragraph.</p>
                                </body>
                            </html>

Предимства на Internal CSS:

  • Могат да се използват класове и ID селектори, за които ще си говорим после. Пример:
    •                                 
                                    .class {
                                        property1 : value1; 
                                        property2 : value2; 
                                        property3 : value3; 
                                    }
                                    
                                    #id {
                                        property1 : value1; 
                                        property2 : value2; 
                                        property3 : value3; 
                                    }
  • Понеже стилът се добавя в същия HTML файл, няма нужда да използваме по няколко файла за един уеб документ.

Недостатъци на Internal CSS:

  • Добавянето на CSS кода към HTML документа може да увеличи размера на документа и неговото зареждане в браузъра.

External CSS:

Чрез External CSS можете да свържете вашата уеб страница към външен .css файл, който съдържа информация за стилизацията и може да бъде създаден чрез всеки текстов редактор на вашето устройство (Notepad, Notepad++, Visual Studio Code и т.н.).

Този метод е най-ефикасен, особено когато се използва за стилизиране на големи уебстраници. Чрез редактиране само на един .css файл, можете лесно да правите промени по целия уебсайт наведнъж.

Пример за използването на External CSS:

  1. Създайте си нов .css файл и дефинирайте някакъв стил на елементите. Пример:
    2.                                 
                                    body {
                                        background-color: orange;
                                        font-family: Helvetica, Arial, Sans-Serif;
                                    }

                                    p {
                                        background-color: white;
                                    }
  2. Добавете в <head> секцията на вашия HTML докуемнт референция към външния .css файл. Това става след тага <title>:
    3.                                 
                                    <link rel="stylesheet" type="text/css" href="style.css" />

Предимства на External CSS:

  • Понеже CSS кодът се намира в отделен документ, HTML файловете са по-добре структурирани, с по-четим код и по-малки като размер.
  • Файлът .css може да се използва многократно за различни страници и документи.

Недостатъци на External CSS:

  • Страниците на HTML документа може да не се визуализират правилно, докато не се зареди целия външен CSS файл.
  • Качването или референцията към няколко CSS файла може да увеличи времето за зареждане.

Синтаксис на CSS

Както вече разбрахме, основната цел на CSS е да придаде специфични свойства на елементите на даден уеб документ - цвят, оформление на текста, позициониране на елементите в пространството и др. CSS правилото се състои от три основни елемента - selector, property и value. Property и value се съдържат в блока с декларацията на даден селектор. Блокът с декларацията съдържа поне една декларация, разделена с ";" и оградена с "{}". Нека разгледаме за какво служи всеки един от елементите [2]:

  • Selector - посочва върху кои HTML елементи да се изпълни стилизацията
  • Propety - идентификатор, който показва какво свойство да бъде приложено
  • Value - стойност на дадено property. Всяко property има набор от валидни стойности, дефинирани от формална граматика, както и семантично значение, реализирано от браузъра
                        
                            p {
                                color: blue;
                                text-align: center;
                            }
Пояснение към примера:
  • p е селектор в CSS и сочи към HTML елемента, който искаме да стилизираме
  • color е property, което показва какво ще стилизираме, а red е неговата стойност (value)
  • text-align е propety, което показва какво ще стилизираме, а center е неговата стойност (value)

Видове селектори в CSS

CSS селекторите дефинират елемента, към който ще се изпълни дадено CSS правило. Различаваме няколко основни типа селектори, които ще разгледаме. [3]

Прости селектори в CSS

Простият селектор се състои от едно условие. Селекторът на тип (type selector), универсалният селектор (universal selector), селекторът на атрибути (attribute selector), селекторът на ID (ID selector), селекторът на класове (class selector) и псевдо-класовете са прости селектори. Сега ще разгледаме отделните типове прости селектори.

Universal selector:

Избира всички елементи. По избор може да бъде ограничено до конкретно пространство от имена или до всички пространства от имена.

Синтаксис: *

Пример: * ще избере всички елементи в документа. 

                                    
                                        * {
                                            background-color: blue;
                                        }

Type selector:

Избира всички елементи от определен тип.

Синтаксис: elementname

Пример: p ще избере всеки <p> елемент. 

                                    
                                        p {
                                            text-align: center;
                                            color: red;
                                        }

Class selector:

Избира всички елементи от даден клас. За да бъдат избрани елементи от определен клас, се използва символът ".", последван от името на класа.

Синтаксис: .classname

Пример: .center ще избере всички елементи от клас "center". 

                                    
                                        .center {
                                            text-align: center;
                                            color: blue;
                                        }

Можем да задаваме и точно определен HTML елемент, който да бъде обхванат от CSS правилото. Това е възможно чрез изписване на името на елемента, след което "." и името на конкретния клас

Пример: 

                                    
                                        p.center {
                                            background-color: yellow;
                                        }

По този начин за всички елементи от тип <p> и притежаващи клас "center", ще оцветим фона в жълто.

ID selector:

Избира елемент спрямо стойността на неговия id атрибут. В документа трябва да има само един елемент с конкретното id.

Синтаксис: #idname

Пример: #example ще избере само този елемент, който притежава id "example". 

                                    
                                        #example {
                                            text-align: center;
                                            color: blue;
                                        }

Attribute selector:

Избира всички елементи, които притежават конкретен атрибут или стойност на атрибут. [4]

[attr] selector:

[attr] селекторът избира елементи по атрибут.

Синтаксис: [attr]

Пример: Избира всички <a> елементи с атрибут "title". 

                                    
                                        a[title] {
                                            color: blue;
                                        }

[attr="value"] selector:

[attr="value"] селекторът избира елементи по конкретни атрибут и стойност.

Синтаксис: [attr="value"]

Пример: Избира всички <a> елементи с атрибут "href", който има стойност "https://example.org". 

                                    
                                        a[href="https://example.org"] {
                                            color: purple;
                                        }

[attr~="value"] selector:

[attr~="value"] селекторът избира елементи по атрибут, чиято стойност съдържа в себе си "value".

Синтаксис: [attr~="value"]

Пример: Избира всички елементи с атрибут "title", чиято стойност съдържа в себе си лист от думи, с разстояние space, и една от тези думи е "flower". 

                                    
                                        [title~="flower"] {
                                            border: 5px solid blue;
                                        }

Забележка: Примерът ще вземе всички елементи, на които title="flower", title="summer flower" и title="flower new", но няма да вземе елементи с title="my-flower" или title="flowers".

[attr|="value"]

[attr|="value"] селекторът избира елементи с определен атрибут, чиято стойност е точно дадената стойност "value" или дадената стойност "value", последвана от тире (-).

Синтаксис: [att|="value"]

Пример: 

                                    
                                        [class|="top"] {
                                            background: blue;
                                        }

Забележка: Сойността трябва да бъде цяла дума, например class="top", или следвана от тире, например class="top-text".

[attr^="value"] selector:

[attr^="value"] селекторът избира елементи с определен атрибут, чиято стойност започва с определено "value".

Синтаксис: [attr^="value"]

Пример: Избира всички елементи с атрибут "class", чиято стойност започва с "top". 

                                    
                                        [class^="top"] {
                                            background: blue;
                                        }

Забележка: Стойността не е задължително да бъде цяла дума.

[attr$="value"] selector:

[attr$="value"] селекторът избира елементи с определен атрибут, чиято стойност завършва с определено "value".

Синтаксис: [attr$="value"]

Пример: Избира всички <a> елементи с атрибут "href", чиято стойност завършва с ".org". 

                                    
                                        a[href$=".org"] {
                                            font-style: italic;
                                        }

Забележка: Стойността не е задължително да бъде цяла дума.

[attr*="value"] selector:

[attr*="value"] селекторът избира елементи с определен атрибут, чиято стойност съдържа някъде в себе си определено "value".

Синтаксис: [attr*="value"]

Пример: Избира всички <a> елементи с атрибут "href", чиято стойност съдържа някъде в себе си "example". 

                                    
                                        a[href*="example"] {
                                            font-size: 2em;
                                        }

Забележка: Стойността не е задължително да бъде цяла дума.

Pseudo-Classes:

Псевдо-класовете ни позволяват да приложим стил към елемент не само по отношение на съдържанието на дървото на документа, но и във връзка с външни фактори, някои от които са: история на навигатора (например :visited), състоянието на съдържанието на даден елемент (например :checked), позицията на мишката (например :hover) и много други. Те са прости селектори, които позволяват избор въз основа на информация, която се намира извън дървото на документа, или която може да бъде невъзможна за изразяване с помощта на другите прости селектори, които вече разгледахме.

Псевдо класовете могат да бъдат два вида - динамични и структурни. Псевдо класовете могат да бъдат динамични в смисъл, че даден елемент може да придобие или да загуби псевдо клас, докато потребителят взаимодейства с документа, без самият документ да се променя. Могат да бъдат структурни в смисъл, че те избират даден елемент спрямо неговата позиция в дървото и връзката му с другите елементи. Псевдо класът не се появява и не променя източника или дървото на документа. [5] Сега ще разгледаме примери за някои от класовете. [6]

Синтаксис: 

                                    
                                        selector:pseudo-class {
                                            property: value;
                                        }

Linguistic pseudo-classes:

Тези псевдо-класове отразяват езика на документа и позволяват избор на елементи въз основа на посоката на езика или скрипта.

  • :dir
  • :lang - избира елемент въз основа на езика на съдържанието му

Location pseudo-classes:

Тези псевдо-класове се отнасят до връзки (линкове).

  • :any-link - избира елемент, който отговаря на :link или на :visited
  • :link - избира всички линкове, които все още не са посетени от потребителя
  • :visited - избира всички линкове, които вече са посетени от потребителя
  • :local-link - избира всички линкове, които сочат към същата страница
  • :target - избира елемента, към който води даден линк
  • :target-within - избира всеки елемент, към който сочи даден линк, и всички елементи, които имат наследник, към който сочи този линк
  • :scope

User action pseudo-classes:

Тези псевдо-класове изискват известно взаимодействие от страна на потребителя, за да бъдат приложени.

  • :hover - избира елемента, който потребителят е посочил с мишката
  • :active - избира елемента, който е бил активиран от потребителя, например чрез кликване върху него
  • :focus - избира елемента, който е на фокус
  • :focus-visible
  • :focus-within - избира елемент, към който е приложен :focus, както и всеки елемент, който има наследник, към който е приложен :focus

Time-dimensional pseudo-classes:

This pseudo-class will select the time-dimensional relative element like during a speech rendering or during the display of HTML5 video subtitles.

  • :current - представя елемента или предшественика на елемента, който се показва
  • :past - представя елемента, който се появява изцяло преди :current елемента
  • :future - представя елемента, който се появява изцяло след :current елемента

Resource state pseudo-classes:

Тези псевдо-класове се прилагат за медии, които могат да бъдат в състояние, в което биха били описани като възпроизвеждащи, като например видео.

  • :playing - избира медиен елемент, който може да се изпълни, когато този елемент се възпроизвежда
  • :paused - избира медиен елемент, който може да се изпълни, когато този елемент е поставен на пауза

Input pseudo-classes:

Тези псевдо-класове се отнасят до form елементите и позволяват избора на елементи въз основа на HTML атрибути и състоянието, в което се намира полето преди и след взаимодействието с него.

  • :autofill - избира елемента, чийто <input> е бил попълнен автоматично от браузъра
  • :enabled - избира елемент, който е в активирано състояние
  • :disabled - избира елемент, който е в деактивирано състояние
  • :read-only - избира елемент, който не може да бъде променян от потребителя
  • :read-write - избира елемент, който може да бъде променян от потребителя
  • :placeholder-shown - избира елемент, който има атрибут placeholder, който показва даден текст
  • :default
  • :checked - избира всеки <input> елемент, който е избран от потребителя
  • :blank - избира всеки <input> елемент, който е празен, съдържа празен стринг или друг нулев вход
  • :valid - избира всеки <input> елемент, който съдържа валидни входни данни
  • :invalid - избира всеки <input> елемент, който съдържа невалидни входни данни
  • :in-range - използва се върху елементи, които имат зададена граница на стойностите, например при плъзгач, когато избраната стойност е в зададения интервал
  • :out-of-range - използва се върху елементи, които имат зададена граница на стойностите, например при плъзгач, когато избраната стойност не е в зададения интервал
  • :required - избира <input> елементи, които са задължителни
  • :optional - избира <input> елементи, които са опционални
  • :user-invalid - избира елемент с некоректни входни данни, но само когато потребителят е взаимодействал с него

Tree-structural pseudo-classes:

Тези псевдо-класове се отнасят до местоположението на елемент в дървото на документа.

  • :root - избира root елемента на документа. В HTML това обикновено е <html> елемента
  • :empty - избира елемент, който няма деца
  • :nth-child - избира n-тото дете на даден елемент
  • :nth-last-child - избира n-toto дете на даден елемент в обратен ред
  • :first-child - избира първото дете на даден елемент
  • :last-child - избира последното дете на даден елемент
  • :only-child - избира елементи, които имат единствено дете, например лист с един единствен list item
  • :nth-of-type - избира n-тия елемент от даден тип в конкретен селектор
  • :nth-last-of-type - избира n-тия елемент от даден тип в конкретен селектор в обратен ред
  • :first-of-type - избира първия срещан елемент от даден тип в конкретен селектор
  • :last-of-type - избира последния срещан елемент от даден тип в конкретен селектор
  • :only-of-type - избира елемент, който е единствен от даден тип в конкретен селектор

Pseudo-Elements:

Псевдо-елементите в CSS са ключови думи, които се добавят към селектор. Това ни позволява да стилизираме конкретна част от избрания елемент. Сега ще разгледаме някои псевдо елементи. [7]

Синтаксис: 

                                    
                                        selector::pseudo-element {
                                            property: value;
                                        }

  • ::first-line - добавя стил към първия ред на даден елемент, може да се използва само върху block-level елементи
  • ::first-letter - добавя стил към първата буква на даден текст, може да се използва само върху block-level елементи
  • ::before - използва се, за да се добави някакво съдържание преди даден елемент
  • ::after - използва се, за да се добави някакво съдържание след даден елемент
  • ::marker - позволява добавянето на стил към маркера на list item-ите в даден лист
  • ::selection - позволява добавянето на стил към полето, което потребителят е маркирал с мишката

Групиращ селектор

Групиращият селектор (Grouping selector) избира всички HTML елементи, които споделят обща стилизация на елементите.

Пример: Нека погледнем следния код. Елементите h1, h2 и p имат една и съща стилизация: 

                                
                                    h1 {
                                        text-align: center;
                                        color: red;
                                    }
                                      
                                    h2 {
                                        text-align: center;
                                        color: red;
                                    }
                                      
                                    p {
                                        text-align: center;
                                        color: red;
                                    }

Можем да използваме групиращия селектор за минимизиране на кода. За да групираме отделните селектори, разделяме всеки селектор със запетая (,).

Пример: 

                                
                                    h1, h2, p {
                                        text-align: center;
                                        color: red;
                                    }

Комбиниращи селектори

Комбинаторът е нещо, което обяснява връзката между селекторите. CSS селекторът може да съдържа няколко прости селектора, между които поставяме комбинатор. Съществуват 4 типа комбинатори [8]:

  • descendant selector (space)
  • child selector (>)
  • adjacent sibling selector (+)
  • general sibling selector (~)
Сега ще разгледаме всеки един от комбинаторите.

Descendant selector:

Комбинаторът " " (space) избира всички елементи, които са наследници на първия елемент.

Синтаксис: A B

Пример: div p ще избере всички <p> елементи, които се намират в <div> елемент. 

                                
                                    div p {
                                        background-color: blue;
                                    }

Забележка: Този комбинатор избира всички елементи с определен родител. Елементите, които не са наследници на този родител, няма да бъдат избрани.

Child selector:

Комбинаторът ">" избира всички елементи, които са преки деца на първия елемент.

Синтаксис: A > B

Пример: div > p ще избере всички <p> елементи, които са преки деца на <div> елемента. 

                                
                                    div > p {
                                        background-color: blue;
                                    }

Забележка: Този комбинатор избира всички елементи с определен родител. Елементите, които не са преки деца на този родител, няма да бъдат избрани.

Adjacent sibling selector:

Комбинаторът "+" избира втория елемент, ако той е разположен директно след първия специфициран елемент.

Синтаксис: A + B

Пример: div + p ще избере първия <p> елемент, който е разположен директно след <div> елемент. 

                                
                                    div + p {
                                        background-color: blue;
                                    }

General sibling selector:

Комбинаторът "~" избира всички елементи, които са siblings. Това означава, че вторият елемент се намира след първия елемент (не е нужно да е непосредствено след него) и двата елемента споделят общ родител.

Синтаксис: A ~ B

Пример: div ~ p ще избере всички <p> елементи, които се намират след <div> елемент. 

                                
                                    div ~ p {
                                        background-color: blue;
                                    }

Заключение

Този реферат има за цел да даде цялостна информация за това какво е CSS, за какво се използва, какъв е синтаксиса му, какви стилове съществуват, както и да представи в дълбочина видовете селектори и с какво те се различават.
